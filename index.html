Modern Web Deisng

Modern web design consists of 3 major front-end languages, HTML,CSS and JavaScript. HTML creates the structure for the project, such as the text on the screen or links you can click on. CSS makes HTML look good, and makes your website unique. JavaScript adds logic to your website, which could mean saving form data, links you to another page when you click a button, or anything that can't be done in HTML.

The Terminal

The terminal gives you direct access to your computer, with no software needed.

GitHub

GitHub is a place where Git repo's can be viewed easily. GitHub makes it easy to share code or work on projects together with a team, or keep a copy of your personal project on the cloud.

Markdown

Markdown is plain-text file which gives you formatting options, including bold, italics, headers and more. It is useful for sites like GitHub, where they can be easily viewed.
